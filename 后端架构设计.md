### 3.2 统一响应封装

```java
@Data
@NoArgsConstructor
@AllArgsConstructor
public class Result<T> implements Serializable {

    private static final long serialVersionUID = 1L;

    /** 状态码 */
    private Integer code;

    /** 返回消息 */
    private String message;

    /** 返回数据 */
    private T data;

    /** 时间戳 */
    private Long timestamp;

    public Result() {
        this.timestamp = System.currentTimeMillis();
    }

    // 成功响应
    public static <T> Result<T> success() {
        return success(null);
    }

    public static <T> Result<T> success(T data) {
        Result<T> result = new Result<>();
        result.setCode(ResultCode.SUCCESS.getCode());
        result.setMessage(ResultCode.SUCCESS.getMessage());
        result.setData(data);
        return result;
    }

    // 失败响应
    public static <T> Result<T> error(String message) {
        return error(ResultCode.ERROR.getCode(), message);
    }

    public static <T> Result<T> error(Integer code, String message) {
        Result<T> result = new Result<>();
        result.setCode(code);
        result.setMessage(message);
        return result;
    }
}

// 响应码枚举
@Getter
@AllArgsConstructor
public enum ResultCode {
    SUCCESS(200, "操作成功"),
    ERROR(500, "操作失败"),
    UNAUTHORIZED(401, "未登录或登录已过期"),
    FORBIDDEN(403, "没有权限访问"),
    NOT_FOUND(404, "资源不存在"),
    PARAM_ERROR(400, "参数错误"),
    ALARM_NOT_FOUND(1001, "预警不存在"),
    DEVICE_OFFLINE(2001, "设备离线"),
    ;

    private final Integer code;
    private final String message;
}
```

### 3.3 全局异常处理

```java
@Slf4j
@RestControllerAdvice
public class GlobalExceptionHandler {

    // 业务异常
    @ExceptionHandler(BusinessException.class)
    public Result<Void> handleBusinessException(BusinessException e) {
        log.warn("业务异常: {}", e.getMessage());
        return Result.error(e.getCode(), e.getMessage());
    }

    // 参数校验异常
    @ExceptionHandler(MethodArgumentNotValidException.class)
    public Result<Void> handleValidException(MethodArgumentNotValidException e) {
        String message = e.getBindingResult().getFieldErrors().stream()
            .map(FieldError::getDefaultMessage)
            .collect(Collectors.joining(", "));
        return Result.error(ResultCode.PARAM_ERROR.getCode(), message);
    }

    // 认证异常
    @ExceptionHandler(AccessDeniedException.class)
    public Result<Void> handleAccessDeniedException(AccessDeniedException e) {
        return Result.error(ResultCode.FORBIDDEN.getCode(), "没有权限执行此操作");
    }

    // 系统异常
    @ExceptionHandler(Exception.class)
    public Result<Void> handleException(Exception e) {
        log.error("系统异常: ", e);
        return Result.error("系统繁忙，请稍后重试");
    }
}
```

### 3.4 AOP日志记录

```java
@Aspect
@Component
@Slf4j
public class LogAspect {

    @Pointcut("@annotation(com.pdi.api.annotation.OperationLog)")
    public void logPointCut() {}

    @Around("logPointCut()")
    public Object around(ProceedingJoinPoint point) throws Throwable {
        long beginTime = System.currentTimeMillis();

        // 执行方法
        Object result = point.proceed();

        // 执行时长
        long time = System.currentTimeMillis() - beginTime;

        // 保存日志
        saveLog(point, time);

        return result;
    }

    private void saveLog(ProceedingJoinPoint joinPoint, long time) {
        MethodSignature signature = (MethodSignature) joinPoint.getSignature();
        Method method = signature.getMethod();
        OperationLog annotation = method.getAnnotation(OperationLog.class);

        OperationLogEntity logEntity = new OperationLogEntity();
        logEntity.setModule(annotation.module());
        logEntity.setOperation(annotation.operation());
        logEntity.setRequestMethod(joinPoint.getTarget().getClass().getName() + "." + method.getName());
        logEntity.setRequestParams(JSON.toJSONString(joinPoint.getArgs()));
        logEntity.setExecuteTime(time);
        logEntity.setCreateTime(LocalDateTime.now());

        // 获取当前用户
        SecurityUser user = SecurityUtils.getCurrentUser();
        if (user != null) {
            logEntity.setUserId(user.getUserId());
            logEntity.setUsername(user.getUsername());
        }

        // 异步保存日志
        logService.saveAsync(logEntity);
    }
}

// 自定义注解
@Target(ElementType.METHOD)
@Retention(RetentionPolicy.RUNTIME)
@Documented
public @interface OperationLog {
    String module() default "";
    String operation() default "";
}
```

### 4.1 Spring Security配置

```java
@Configuration
@EnableWebSecurity
@EnableMethodSecurity(prePostEnabled = true)
public class SecurityConfig {

    @Autowired
    private JwtAuthenticationFilter jwtAuthenticationFilter;

    @Autowired
    private AuthenticationEntryPoint authenticationEntryPoint;

    @Autowired
    private AccessDeniedHandler accessDeniedHandler;

    @Bean
    public SecurityFilterChain filterChain(HttpSecurity http) throws Exception {
        http
            // 禁用CSRF（使用JWT）
            .csrf(csrf -> csrf.disable())

            // 配置授权规则
            .authorizeHttpRequests(auth -> auth
                .requestMatchers("/api/auth/login", "/api/auth/refresh").permitAll()
                .requestMatchers("/api/receiver/**").permitAll()  // 数据接收接口
                .requestMatchers("/ws/**").permitAll()  // WebSocket
                .requestMatchers("/actuator/**").hasRole("ADMIN")
                .anyRequest().authenticated()
            )

            // 禁用session
            .sessionManagement(session ->
                session.sessionCreationPolicy(SessionCreationPolicy.STATELESS)
            )

            // 添加JWT过滤器
            .addFilterBefore(jwtAuthenticationFilter,
                UsernamePasswordAuthenticationFilter.class)

            // 异常处理
            .exceptionHandling(ex -> ex
                .authenticationEntryPoint(authenticationEntryPoint)
                .accessDeniedHandler(accessDeniedHandler)
            );

        return http.build();
    }

    @Bean
    public PasswordEncoder passwordEncoder() {
        return new BCryptPasswordEncoder();
    }

    @Bean
    public AuthenticationManager authenticationManager(
            AuthenticationConfiguration config) throws Exception {
        return config.getAuthenticationManager();
    }
}
```

### 4.2 JWT认证流程

```java
@Component
public class JwtTokenProvider {

    @Value("${jwt.secret}")
    private String jwtSecret;

    @Value("${jwt.expiration:7200000}")  // 2小时
    private long jwtExpiration;

    @Value("${jwt.refresh-expiration:604800000}")  // 7天
    private long refreshExpiration;

    private SecretKey key;

    @PostConstruct
    public void init() {
        this.key = Keys.hmacShaKeyFor(jwtSecret.getBytes(StandardCharsets.UTF_8));
    }

    // 生成访问令牌
    public String generateToken(SecurityUser user) {
        Date expiryDate = new Date(System.currentTimeMillis() + jwtExpiration);

        return Jwts.builder()
            .subject(user.getUserId().toString())
            .claim("username", user.getUsername())
            .claim("role", user.getRole())
            .claim("siteId", user.getSiteId())
            .issuedAt(new Date())
            .expiration(expiryDate)
            .signWith(key)
            .compact();
    }

    // 生成刷新令牌
    public String generateRefreshToken(SecurityUser user) {
        Date expiryDate = new Date(System.currentTimeMillis() + refreshExpiration);

        return Jwts.builder()
            .subject(user.getUserId().toString())
            .claim("type", "refresh")
            .issuedAt(new Date())
            .expiration(expiryDate)
            .signWith(key)
            .compact();
    }

    // 验证并解析令牌
    public Claims validateToken(String token) {
        try {
            return Jwts.parser()
                .verifyWith(key)
                .build()
                .parseSignedClaims(token)
                .getPayload();
        } catch (ExpiredJwtException e) {
            throw new BusinessException("令牌已过期");
        } catch (JwtException e) {
            throw new BusinessException("无效令牌");
        }
    }

    // 从令牌获取用户ID
    public Long getUserIdFromToken(String token) {
        Claims claims = validateToken(token);
        return Long.parseLong(claims.getSubject());
    }
}

// JWT过滤器
@Component
public class JwtAuthenticationFilter extends OncePerRequestFilter {

    @Autowired
    private JwtTokenProvider tokenProvider;

    @Autowired
    private UserDetailsService userDetailsService;

    @Override
    protected void doFilterInternal(HttpServletRequest request,
            HttpServletResponse response, FilterChain filterChain)
            throws ServletException, IOException {

        String token = getTokenFromRequest(request);

        if (StringUtils.hasText(token)) {
            try {
                Claims claims = tokenProvider.validateToken(token);
                String userId = claims.getSubject();

                UserDetails userDetails = userDetailsService.loadUserById(Long.parseLong(userId));

                UsernamePasswordAuthenticationToken authentication =
                    new UsernamePasswordAuthenticationToken(
                        userDetails, null, userDetails.getAuthorities());
                authentication.setDetails(
                    new WebAuthenticationDetailsSource().buildDetails(request));

                SecurityContextHolder.getContext().setAuthentication(authentication);
            } catch (Exception e) {
                logger.error("无法设置用户认证: {}", e.getMessage());
            }
        }

        filterChain.doFilter(request, response);
    }

    private String getTokenFromRequest(HttpServletRequest request) {
        String bearerToken = request.getHeader("Authorization");
        if (StringUtils.hasText(bearerToken) && bearerToken.startsWith("Bearer ")) {
            return bearerToken.substring(7);
        }
        return null;
    }
}
```

```java
// 数据权限注解
@Target({ElementType.METHOD, ElementType.TYPE})
@Retention(RetentionPolicy.RUNTIME)
@PreAuthorize("hasRole('ADMIN') or @dataPermissionChecker.check(#root)")
public @interface DataPermission {
    String value() default "";
}

// 数据权限检查器
@Component("dataPermissionChecker")
public class DataPermissionChecker {

    public boolean check(JoinPoint joinPoint) {
        SecurityUser currentUser = SecurityUtils.getCurrentUser();
        if (currentUser == null) {
            return false;
        }

        // 超级管理员拥有所有权限
        if (currentUser.isSuperAdmin()) {
            return true;
        }

        // 从参数中获取站点ID
        Object[] args = joinPoint.getArgs();
        for (Object arg : args) {
            if (arg instanceof DataScope) {
                Long siteId = ((DataScope) arg).getSiteId();
                if (siteId != null) {
                    return currentUser.getSiteId().equals(siteId);
                }
            }
        }

        return true;
    }
}
```

### 4.4 登录安全

```java
@Service
public class AuthServiceImpl implements AuthService {

    @Autowired
    private AuthenticationManager authenticationManager;

    @Autowired
    private JwtTokenProvider tokenProvider;

    @Autowired
    private UserService userService;

    @Autowired
    private RedisTemplate<String, Object> redisTemplate;

    private static final String LOGIN_FAIL_KEY = "login:fail:";
    private static final int MAX_FAIL_COUNT = 5;
    private static final long LOCK_TIME = 30;  // 30分钟

    @Override
    public LoginVO login(LoginDTO loginDTO) {
        String username = loginDTO.getUsername();
        String ip = IpUtils.getIpAddr();

        // 检查是否被锁定
        String lockKey = LOGIN_FAIL_KEY + username;
        Integer failCount = (Integer) redisTemplate.opsForValue().get(lockKey);
        if (failCount != null && failCount >= MAX_FAIL_COUNT) {
            throw new BusinessException("登录失败次数过多，请" + LOCK_TIME + "分钟后重试");
        }

        try {
            // 认证
            Authentication authentication = authenticationManager.authenticate(
                new UsernamePasswordAuthenticationToken(
                    username, loginDTO.getPassword())
            );

            SecurityUser user = (SecurityUser) authentication.getPrincipal();

            // 检查用户状态
            if (user.getStatus() != 1) {
                throw new BusinessException("账号已被禁用");
            }

            // 生成令牌
            String accessToken = tokenProvider.generateToken(user);
            String refreshToken = tokenProvider.generateRefreshToken(user);

            // 清除失败计数
            redisTemplate.delete(lockKey);

            // 记录登录日志
            saveLoginLog(user, ip, true);

            return LoginVO.builder()
                .accessToken(accessToken)
                .refreshToken(refreshToken)
                .expiresIn(tokenProvider.getExpiration())
                .user(convertToUserVO(user))
                .build();

        } catch (BadCredentialsException e) {
            // 增加失败计数
            failCount = (failCount == null) ? 1 : failCount + 1;
            redisTemplate.opsForValue().set(lockKey, failCount, LOCK_TIME, TimeUnit.MINUTES);

            saveLoginLog(null, ip, false);
            throw new BusinessException("用户名或密码错误，还剩" + (MAX_FAIL_COUNT - failCount) + "次机会");
        }
    }
}
```

```java
@Service
public class AlarmServiceImpl extends ServiceImpl<AlarmMapper, AlarmEntity>
        implements AlarmService {

    @Autowired
    private WebSocketService webSocketService;

    @Autowired
    private RedisTemplate<String, Object> redisTemplate;

    @Autowired
    private AlarmHandlerChain handlerChain;

    private static final String ALARM_CACHE_KEY = "alarm:realtime:";

    @Override
    @Transactional
    public AlarmVO createAlarm(AlarmCreateDTO dto) {
        // 重复预警检查（5分钟内同一设备同类型不重复生成）
        String dupKey = ALARM_CACHE_KEY + dto.getDeviceId() + ":" + dto.getAlarmType();
        if (Boolean.TRUE.equals(redisTemplate.hasKey(dupKey))) {
            return null;
        }

        AlarmEntity alarm = new AlarmEntity();
        BeanUtils.copyProperties(dto, alarm);
        alarm.setAlarmTime(LocalDateTime.now());
        alarm.setStatus(AlarmStatus.UNHANDLED.getCode());
        alarm.setCreateTime(LocalDateTime.now());

        save(alarm);

        // 设置重复预警缓存
        redisTemplate.opsForValue().set(dupKey, alarm.getId(), 5, TimeUnit.MINUTES);

        AlarmVO alarmVO = convertToVO(alarm);

        // 异步推送
        pushAlarm(alarmVO);

        return alarmVO;
    }

    @Override
    @Async
    public void pushAlarm(AlarmVO alarm) {
        // WebSocket推送
        webSocketService.sendAlarm(alarm);

        // 触发处理器链（短信、邮件等）
        handlerChain.handle(alarm);
    }

    @Override
    public PageResult<AlarmVO> listRealTimeAlarms(AlarmQueryDTO query) {
        LambdaQueryWrapper<AlarmEntity> wrapper = new LambdaQueryWrapper<>();
        wrapper.eq(AlarmEntity::getStatus, AlarmStatus.UNHANDLED.getCode());

        // 数据权限过滤
        applyDataPermission(wrapper, query);
```

```java
            System.currentTimeMillis() + ".zip";

             ZipOutputStream zos = new ZipOutputStream(baos)) {

                        .bucket("pdi-images")
                        .object(image.getObjectName())
                        .build()
                );

                zos.putNextEntry(entry);
                IOUtils.copy(is, zos);

                    .bucket("pdi-packages")
                    .object(zipFileName)
                    .stream(bais, baos.size(), -1)
                    .contentType("application/zip")
                    .build()
            );

            return minioClient.getPresignedObjectUrl(
                GetPresignedObjectUrlArgs.builder()
                    .method(Method.GET)
                    .bucket("pdi-packages")
                    .object(zipFileName)
                    .expiry(3600)
                    .build()
            );
```

```java
/**
 * PDI作业状态机
 * 基础状态三元组: S = (door_open, person_present, person_entering_exiting)
 */
@Component
public class PdiStateMachine {

    @Autowired
    private StateMachine<PDIState, PDIEvent> stateMachine;

    @Autowired
    private ArchiveService archiveService;

    @Autowired
    private RedisTemplate<String, Object> redisTemplate;

    private static final String STATE_KEY = "pdi:state:";
    private static final String SEQUENCE_KEY = "pdi:sequence:";

    /**
     * 处理状态流转
     */
    public void handleStateTransition(Long channelId, DeviceStateDTO newState) {
        String stateKey = STATE_KEY + channelId;
        String seqKey = SEQUENCE_KEY + channelId;

        // 获取当前状态
        PDIState currentState = (PDIState) redisTemplate.opsForValue().get(stateKey);
        if (currentState == null) {
            currentState = PDIState.IDLE;
        }

        // 解析新状态
        PDIState targetState = parseState(newState);

        // 如果状态未变化，直接返回
        if (currentState == targetState) {
            return;
        }

        // 获取状态转换事件
        PDIEvent event = determineEvent(currentState, targetState);
        if (event == null) {
            log.warn("无效的状态转换: {} -> {}", currentState, targetState);
            return;
        }

        // 执行状态转换
        StateContext<PDIState, PDIEvent> context =
            stateMachine.sendEvent(Mono.just(MessageBuilder
                .withPayload(event)
                .setHeader("channelId", channelId)
                .setHeader("state", newState)
                .build()))
            .blockLast();

        // 保存新状态
        redisTemplate.opsForValue().set(stateKey, targetState, 1, TimeUnit.HOURS);

        // 记录状态序列
        recordStateSequence(channelId, targetState, seqKey);

        // 检查进入/离开判定
        checkEnterOrExit(channelId, seqKey, targetState);
    }

    /**
     * 解析设备状态为PDI状态
     */
    private PDIState parseState(DeviceStateDTO state) {
        boolean doorOpen = state.getDoorOpen();
        boolean personPresent = state.getPersonPresent();
        boolean personEnteringExiting = state.getPersonEnteringExiting();

        if (!doorOpen && !personPresent && !personEnteringExiting) {
            return PDIState.IDLE;  // S1 = (0, 0, 0)
        }
        if (!doorOpen && personPresent && !personEnteringExiting) {
            return PDIState.CLOSED_WITH_PERSON;  // S3 = (0, 1, 0)
        }
        if (doorOpen && !personPresent && !personEnteringExiting) {
            return PDIState.OPEN_NO_PERSON;  // S5 = (1, 0, 0)
        }
        if (doorOpen && personPresent && !personEnteringExiting) {
            return PDIState.OPEN_WITH_PERSON;  // S7 = (1, 1, 0)
        }
        if (doorOpen && personPresent && personEnteringExiting) {
            return PDIState.ENTERING_EXITING;  // S8 = (1, 1, 1)
        }

        return PDIState.UNKNOWN;
    }

    /**
     * 记录状态序列
     */
    private void recordStateSequence(Long channelId, PDIState state, String seqKey) {
        List<PDIState> sequence = (List<PDIState>) redisTemplate.opsForValue().get(seqKey);
        if (sequence == null) {
            sequence = new ArrayList<>();
        }

        sequence.add(state);

        // 只保留最近10个状态
        if (sequence.size() > 10) {
            sequence = sequence.subList(sequence.size() - 10, sequence.size());
        }

        redisTemplate.opsForValue().set(seqKey, sequence, 1, TimeUnit.HOURS);
    }

    /**
     * 检查进入/离开判定
     * 进入判定: [S7 → S8 → S3]
     * 离开判定: [S3 → S8 → S7 → (S5或S1)]
     */
    private void checkEnterOrExit(Long channelId, String seqKey, PDIState currentState) {
        List<PDIState> sequence = (List<PDIState>) redisTemplate.opsForValue().get(seqKey);
        if (sequence == null || sequence.size() < 3) {
            return;
        }

        String archiveKey = "pdi:archive:" + channelId;

        // 检查进入序列: S7 -> S8 -> S3
        if (sequence.size() >= 3) {
            int size = sequence.size();
            if (sequence.get(size - 3) == PDIState.OPEN_WITH_PERSON &&
                sequence.get(size - 2) == PDIState.ENTERING_EXITING &&
                sequence.get(size - 1) == PDIState.CLOSED_WITH_PERSON) {

                // 人员进入，开始PDI作业
                Long archiveId = startPdiArchive(channelId);
                redisTemplate.opsForValue().set(archiveKey, archiveId, 2, TimeUnit.HOURS);

                log.info("PDI作业开始, channelId={}, archiveId={}", channelId, archiveId);
            }
        }

        // 检查离开序列: S3 -> S8 -> S7 -> (S5或S1)
        if (sequence.size() >= 4) {
            int size = sequence.size();
            if (sequence.get(size - 4) == PDIState.CLOSED_WITH_PERSON &&
                sequence.get(size - 3) == PDIState.ENTERING_EXITING &&
                sequence.get(size - 2) == PDIState.OPEN_WITH_PERSON &&
                (sequence.get(size - 1) == PDIState.OPEN_NO_PERSON ||
                 sequence.get(size - 1) == PDIState.IDLE)) {

                // 人员离开，结束PDI作业
                Long archiveId = (Long) redisTemplate.opsForValue().get(archiveKey);
                if (archiveId != null) {
                    completePdiArchive(archiveId);
                    redisTemplate.delete(archiveKey);

                    log.info("PDI作业结束, channelId={}, archiveId={}", channelId, archiveId);
                }
            }
        }
    }

    private Long startPdiArchive(Long channelId) {
        ArchiveCreateDTO dto = new ArchiveCreateDTO();
        dto.setChannelId(channelId);
        // ... 设置其他属性

        ArchiveVO archive = archiveService.createArchive(dto);
        return archive.getId();
    }

    private void completePdiArchive(Long archiveId) {
        ArchiveCompleteDTO dto = new ArchiveCompleteDTO();
        // ... 设置完成信息

        archiveService.completeArchive(archiveId, dto);
    }
}
```

```java
        if (archiveId == null) {
            return;
        }

        // 检查作业时长

        if (startTime != null) {
            Duration duration = Duration.between(startTime, LocalDateTime.now());
            long minutes = duration.toMinutes();

            // 作业时间过长报警
            if (minutes > 120) {
                AlarmCreateDTO alarm = new AlarmCreateDTO();
                alarm.setChannelId(channelId);
                alarm.setAlarmType(AlarmType.PDI_TIMEOUT.getCode());
                alarm.setAlarmLevel(AlarmLevel.MEDIUM.getCode());
                alarm.setDescription("PDI作业时间过长: " + minutes + "分钟");
                alarm.setRelatedArchiveId(archiveId);

                alarmService.createAlarm(alarm);
            }
        }
    }

    /**
     * 异常停留检测
     */
    private void evaluateAbnormalStay(Long channelId, DeviceStateDTO state,
            EventContext context) {
        if (!state.getPersonPresent()) {
            return;
        }

        // 获取人员进入时间
        LocalDateTime enterTime = context.getPersonEnterTime();
        if (enterTime == null) {
            return;
        }

        Duration stayDuration = Duration.between(enterTime, LocalDateTime.now());

        // 非作业时间异常停留（超过30分钟）
        if (stayDuration.toMinutes() > 30 && !context.isPdiInProgress()) {
            AlarmCreateDTO alarm = new AlarmCreateDTO();
            alarm.setChannelId(channelId);
            alarm.setAlarmType(AlarmType.ABNORMAL_STAY.getCode());
            alarm.setAlarmLevel(AlarmLevel.LOW.getCode());
            alarm.setDescription("检测到非作业时间异常停留: " + stayDuration.toMinutes() + "分钟");

            alarmService.createAlarm(alarm);
        }
    }

    /**
     * 门长时间未关检测
     */
    private void evaluateDoorOpenTooLong(Long channelId, DeviceStateDTO state,
            EventContext context) {
        if (!state.getDoorOpen()) {
            return;
        }

        LocalDateTime doorOpenTime = context.getDoorOpenTime();
        if (doorOpenTime == null) {
            return;
        }

        Duration openDuration = Duration.between(doorOpenTime, LocalDateTime.now());

        // 门打开超过10分钟报警
        if (openDuration.toMinutes() > 10) {
            AlarmCreateDTO alarm = new AlarmCreateDTO();
            alarm.setChannelId(channelId);
            alarm.setAlarmType(AlarmType.DOOR_OPEN_TOO_LONG.getCode());
            alarm.setAlarmLevel(AlarmLevel.LOW.getCode());
            alarm.setDescription("门打开时间过长: " + openDuration.toMinutes() + "分钟");

            alarmService.createAlarm(alarm);
        }
    }
```

```java
@Component
@Slf4j
public class WebSocketSessionManager {

    // 存储用户会话
    private final ConcurrentHashMap<String, Set<String>> userSessions = new ConcurrentHashMap<>();

    // 存储会话最后活跃时间
    private final ConcurrentHashMap<String, Long> sessionLastActive = new ConcurrentHashMap<>();

    private static final long HEARTBEAT_TIMEOUT = 60000;  // 60秒

    /**
     * 注册会话
     */
    public void registerSession(String userId, String sessionId) {
        userSessions.computeIfAbsent(userId, k -> ConcurrentHashMap.newKeySet()).add(sessionId);
        sessionLastActive.put(sessionId, System.currentTimeMillis());
        log.info("WebSocket会话注册: userId={}, sessionId={}", userId, sessionId);
    }

    /**
     * 注销会话
     */
    public void unregisterSession(String userId, String sessionId) {
        Set<String> sessions = userSessions.get(userId);
        if (sessions != null) {
            sessions.remove(sessionId);
            if (sessions.isEmpty()) {
                userSessions.remove(userId);
            }
        }
        sessionLastActive.remove(sessionId);
        log.info("WebSocket会话注销: userId={}, sessionId={}", userId, sessionId);
    }

    /**
     * 更新会话活跃时间
     */
    public void updateActiveTime(String sessionId) {
        sessionLastActive.put(sessionId, System.currentTimeMillis());
    }

    /**
     * 获取用户在线状态
     */
    public boolean isUserOnline(String userId) {
        Set<String> sessions = userSessions.get(userId);
        return sessions != null && !sessions.isEmpty();
    }

    /**
     * 获取在线用户数量
     */
    public int getOnlineUserCount() {
        return userSessions.size();
    }

    /**
     * 清理超时会话
     */
    @Scheduled(fixedRate = 30000)  // 每30秒检查一次
    public void cleanTimeoutSessions() {
        long now = System.currentTimeMillis();
        List<String> timeoutSessions = sessionLastActive.entrySet().stream()
            .filter(entry -> now - entry.getValue() > HEARTBEAT_TIMEOUT)
            .map(Map.Entry::getKey)
            .collect(Collectors.toList());

        for (String sessionId : timeoutSessions) {
            sessionLastActive.remove(sessionId);
            // 从用户会话中移除
            userSessions.forEach((userId, sessions) -> sessions.remove(sessionId));
            log.warn("WebSocket会话超时清理: sessionId={}", sessionId);
        }
    }
}
```

```java
@Service
public class BatchInsertService {

    @Autowired
    private SqlSessionFactory sqlSessionFactory;

    /**
     * 批量插入状态数据
     */
    public void batchInsertStatus(List<StatusStreamEntity> list) {
        if (CollectionUtils.isEmpty(list)) {
            return;
        }

        SqlSession session = sqlSessionFactory.openSession(ExecutorType.BATCH, false);
        try {
            StatusStreamMapper mapper = session.getMapper(StatusStreamMapper.class);

            int batchSize = 500;
            for (int i = 0; i < list.size(); i++) {
                mapper.insert(list.get(i));

                if (i % batchSize == 0 || i == list.size() - 1) {
                    session.commit();
                    session.clearCache();
                }
            }
        } finally {
            session.close();
        }
    }

    /**
     * 使用Copy方式批量插入（PostgreSQL）
     */
    public void copyInsertStatus(List<StatusStreamEntity> list) {
        if (CollectionUtils.isEmpty(list)) {
            return;
        }

        String sql = "COPY pdi_status_stream (device_id, door_open, person_present, " +
            "person_entering_exiting, timestamp, create_time) FROM STDIN WITH CSV";

        jdbcTemplate.execute((ConnectionCallback<Void>) conn -> {
            CopyManager copyManager = new CopyManager((BaseConnection) conn);
            StringBuilder sb = new StringBuilder();
```

## 13. 开发规范

### 13.1 代码规范

1. **命名规范**
   - 类名：大驼峰，如 `AlarmServiceImpl`
   - 方法名：小驼峰，如 `createAlarm`
   - 常量：全大写下划线，如 `MAX_RETRY_COUNT`
   - 数据库字段：下划线命名，如 `alarm_time`

2. **分层规范**
   - Controller：只处理请求参数和响应，不包含业务逻辑
   - Service：处理业务逻辑，可调用多个Mapper
   - Mapper：只进行数据访问，不处理业务逻辑

3. **异常规范**
   - 业务异常使用 `BusinessException`
   - 系统异常统一捕获处理
   - 异常信息要具体，便于排查问题

### 13.2 日志规范

```java
// 使用SLF4J
@Slf4j
public class SomeService {

    public void someMethod() {
        // DEBUG：详细的调试信息
        log.debug("处理开始, param={}", param);

        // INFO：重要的业务信息
        log.info("用户 {} 创建了预警 {}", userId, alarmId);

        // WARN：警告信息，不影响主流程
        log.warn("设备 {} 心跳超时", deviceId);

        // ERROR：错误信息，需要处理
        log.error("保存数据失败", exception);
    }
}
```
