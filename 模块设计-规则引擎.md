## 1. 模块职责简述

规则引擎模块是PDI智能监测平台的核心业务处理模块，负责处理来自边缘盒子的状态流数据，实现PDI作业的智能判定与报警。

### 1.1 核心职责

| 职责 | 说明 |
|------|------|
| **状态机管理** | 维护每个通道(channelId)的独立状态机实例，跟踪当前状态和历史状态序列 |
| **状态序列匹配** | 识别人员进入序列(S7→S8→S3)和离开序列(S3→S8→S7→S5/S1) |
| **PDI作业时长计算** | 计算单次作业时长，支持多车门合并计算 |
| **事件触发** | 在状态转换满足规则时触发作业开始/结束事件 |
| **报警判定** | 根据作业时长与标准工时比较，判定是否生成PDI违规报警 |

### 1.2 依赖关系

```
pdi-rule-engine
    ├── 依赖: pdi-common (公共基础模块)
    ├── 被依赖: pdi-algorithm-inlet (算法数据入口模块)
    ├── 被依赖: pdi-alarm-center (预警中心模块)
    └── 被依赖: pdi-behavior-archive (行为档案模块)
```

## 2. 状态机详细设计

### 2.1 状态定义

状态基于三元组 `S = (door_open, person_present, entering_exiting)` 定义：

```java

 * PDI状态枚举

    S3(3, false, true, false, "门关+有人"),
```

### 2.2 状态转换规则

#### 2.2.1 有效状态转换表

| 当前状态 | 可转换至 | 触发条件 |
|---------|---------|---------|
| S1 | S5, S7 | 门开事件 |
| S3 | S8 | 门开+进出中检测 |
| S5 | S1, S7 | 门关/人员出现 |
| S7 | S5, S8 | 人员消失/进出中检测 |
| S8 | S3, S7 | 门关/进出完成 |

#### 2.2.2 进入判定规则 (Entry Rule)

```
触发条件: 检测到状态序列 [S7 → S8 → S3]

时间线示意:
─────────────────────────────────────────────────→
    │      │      │
    S7     S8     S3
   (准备) (进出) (进入完成)
    │      │      │
    └──────┴──────┘
       人员进入车内

记录: entry_time = S3.timestamp
动作: 开始累积车内作业时间，创建PDI作业记录
```

#### 2.2.3 离开判定规则 (Exit Rule)

```
触发条件: 检测到状态序列 [S3 → S8 → S7 → (S5或S1)]

时间线示意:
─────────────────────────────────────────────────→
    │      │      │       │
    S3     S8     S7      S5/S1
   (车内) (进出) (已出)   (离开)
    │      │      │       │
    └──────┴──────┴───────┘
        人员离开流程

记录: exit_time = S5/S1.timestamp
动作: 结束作业计时，计算作业时长，判定合规性
```

### 2.3 StateMachine类设计

```java
/**
 * 状态机实例
 * 每个channelId对应一个独立的StateMachine实例
 */
@Component
@Scope("prototype")
public class StateMachine {

    /** 通道ID */
    private Long channelId;

    /** 站点ID */
    private Long siteId;

    /** 当前状态 */
    private State currentState;

    /** 历史状态序列 (带时间戳) */
    private final LinkedList<StateSnapshot> stateHistory;

    /** 历史记录最大保留数量 */
    private static final int MAX_HISTORY_SIZE = 100;

    /** 当前PDI作业记录ID (进行中时有效) */
    private Long currentTaskId;

    /** 最后更新时间 */
    private LocalDateTime lastUpdateTime;

    /**
     * 状态快照
     */
    @Data
    public static class StateSnapshot {
        private State state;
        private LocalDateTime timestamp;
        private Long stateStreamId;  // 关联原始状态流记录
    }

    /**
     * 初始化状态机
     */
    public void initialize(Long channelId, Long siteId) {
        this.channelId = channelId;
        this.siteId = siteId;
        this.currentState = State.S1;  // 默认空闲状态
        this.stateHistory.clear();
        this.currentTaskId = null;
        this.lastUpdateTime = LocalDateTime.now();
    }

    /**
     * 状态转换
     * @param newState 新状态
     * @param timestamp 状态发生时间
     * @param stateStreamId 原始状态流ID
     * @return 转换结果，包含是否成功及触发的事件
     */
    public StateTransitionResult transition(State newState,
                                            LocalDateTime timestamp,
                                            Long stateStreamId) {
        // 验证转换是否有效
        if (!isValidTransition(currentState, newState)) {
            return StateTransitionResult.invalid(currentState, newState);
        }

        // 记录历史状态
        StateSnapshot snapshot = new StateSnapshot();
        snapshot.setState(newState);
        snapshot.setTimestamp(timestamp);
        snapshot.setStateStreamId(stateStreamId);
        addToHistory(snapshot);

        // 执行状态转换
        State oldState = this.currentState;
        this.currentState = newState;
        this.lastUpdateTime = timestamp;

        // 检查是否触发规则
        RuleTriggerEvent event = checkRuleTrigger(oldState, newState, timestamp);

        return StateTransitionResult.success(oldState, newState, event);
    }

    /**
     * 验证状态转换是否有效
     */
    private boolean isValidTransition(State from, State to) {
        if (from == to) return true;  // 相同状态允许重复

        return switch (from) {
            case S1 -> to == S5 || to == S7;
            case S3 -> to == S8;
            case S5 -> to == S1 || to == S7;
            case S7 -> to == S5 || to == S8;
            case S8 -> to == S3 || to == S7;
        };
    }

    /**
     * 添加状态到历史记录
     */
    private void addToHistory(StateSnapshot snapshot) {
        stateHistory.addLast(snapshot);
        if (stateHistory.size() > MAX_HISTORY_SIZE) {
            stateHistory.removeFirst();
        }
    }

    /**
     * 获取最近N个状态序列
     */
    public List<StateSnapshot> getRecentStates(int n) {
        int size = Math.min(n, stateHistory.size());
        return stateHistory.subList(stateHistory.size() - size, stateHistory.size());
    }

    /**
     * 检查是否匹配进入序列 (S7→S8→S3)
     */
    public boolean matchesEntrySequence() {
        if (stateHistory.size() < 3) return false;

        List<StateSnapshot> recent = getRecentStates(3);
        return recent.get(0).getState() == State.S7 &&
               recent.get(1).getState() == State.S8 &&
               recent.get(2).getState() == State.S3;
    }

    /**
     * 检查是否匹配离开序列 (S3→S8→S7→S5/S1)
     */
    public boolean matchesExitSequence() {
        if (stateHistory.size() < 4) return false;

        List<StateSnapshot> recent = getRecentStates(4);
        State lastState = recent.get(3).getState();

        return recent.get(0).getState() == State.S3 &&
               recent.get(1).getState() == State.S8 &&
               recent.get(2).getState() == State.S7 &&
               (lastState == State.S5 || lastState == State.S1);
    }

    /**
     * 获取进入时间点 (S3状态时间)
     */
    public LocalDateTime getEntryTime() {
        if (!matchesEntrySequence()) return null;
        List<StateSnapshot> recent = getRecentStates(3);
        return recent.get(2).getTimestamp();  // S3的时间戳
    }

    /**
     * 获取离开时间点 (S5或S1状态时间)
     */
    public LocalDateTime getExitTime() {
        if (!matchesExitSequence()) return null;
        List<StateSnapshot> recent = getRecentStates(4);
        return recent.get(3).getTimestamp();  // S5/S1的时间戳
    }
}
```

### 3.2 StateMachineManager (状态机管理器)

```java
/**
 * 状态机管理器
 * 负责按channelId管理状态机实例的生命周期
 */
@Service
@Slf4j
public class StateMachineManager {

    /** channelId -> StateMachine 映射表 */
    private final ConcurrentHashMap<String, StateMachine> stateMachineMap;

    /** 状态机对象池 */
    private final ObjectPool<StateMachine> stateMachinePool;

    /** 状态机最后访问时间记录 (用于清理) */
    private final ConcurrentHashMap<String, LocalDateTime> lastAccessTimeMap;

    /** 状态机空闲超时时间 (分钟) */
    @Value("${rule-engine.state-machine.idle-timeout:30}")
    private int idleTimeoutMinutes;

    private final ApplicationEventPublisher eventPublisher;
    private final StateMachineFactory stateMachineFactory;

    public StateMachineManager(ApplicationEventPublisher eventPublisher,
                               StateMachineFactory stateMachineFactory) {
        this.eventPublisher = eventPublisher;
        this.stateMachineFactory = stateMachineFactory;
        this.stateMachineMap = new ConcurrentHashMap<>();
        this.lastAccessTimeMap = new ConcurrentHashMap<>();
        this.stateMachinePool = new GenericObjectPool<>(new StateMachinePooledFactory());
    }

    /**
     * 获取或创建状态机实例
     */
    public StateMachine getOrCreateStateMachine(Long channelId, Long siteId) {
        String key = generateKey(channelId, siteId);

        StateMachine stateMachine = stateMachineMap.computeIfAbsent(key, k -> {
            try {
                StateMachine sm = stateMachinePool.borrowObject();
                sm.initialize(channelId, siteId);
                log.info("创建状态机实例: channelId={}, siteId={}", channelId, siteId);
                return sm;
            } catch (Exception e) {
                log.error("创建状态机失败", e);
                throw new RuleEngineException("创建状态机失败", e);
            }
        });

        lastAccessTimeMap.put(key, LocalDateTime.now());
        return stateMachine;
    }

    /**
     * 处理状态流事件
     */
    @Transactional
    public void processStateStream(StateStreamEvent event) {
        Long channelId = event.getChannelId();
        Long siteId = event.getSiteId();
        State newState = State.fromCode(event.getStateCode());

        StateMachine stateMachine = getOrCreateStateMachine(channelId, siteId);

        // 执行状态转换
        StateTransitionResult result = stateMachine.transition(
            newState,
            event.getTimestamp(),
            event.getStateStreamId()
        );

        if (!result.isValid()) {
            log.warn("无效状态转换: channelId={}, from={}, to={}",
                channelId, result.getFromState(), result.getToState());
            return;
        }

        // 处理规则触发事件
        if (result.getTriggerEvent() != null) {
            handleRuleTrigger(stateMachine, result.getTriggerEvent());
        }
    }

    /**
     * 处理规则触发
     */
    private void handleRuleTrigger(StateMachine stateMachine, RuleTriggerEvent event) {
        switch (event.getType()) {
            case ENTRY_DETECTED -> handleEntryDetected(stateMachine, event);
            case EXIT_DETECTED -> handleExitDetected(stateMachine, event);
        }
    }

    /**
     * 处理人员进入检测
     */
    private void handleEntryDetected(StateMachine stateMachine, RuleTriggerEvent event) {
        TaskStartEvent startEvent = new TaskStartEvent(
            stateMachine.getChannelId(),
            stateMachine.getSiteId(),
            event.getTimestamp(),
            stateMachine.getCurrentTaskId()
        );
        eventPublisher.publishEvent(startEvent);
    }

    /**
     * 处理人员离开检测
     */
    private void handleExitDetected(StateMachine stateMachine, RuleTriggerEvent event) {
        TaskEndEvent endEvent = new TaskEndEvent(
            stateMachine.getChannelId(),
            stateMachine.getSiteId(),
            event.getTimestamp(),
            stateMachine.getCurrentTaskId(),
            event.getEntryTime()
        );
        eventPublisher.publishEvent(endEvent);
    }

    /**
     * 清理空闲状态机 (定时任务)
     */
    @Scheduled(fixedDelay = 600000) // 每10分钟执行
    public void cleanupIdleStateMachines() {
        LocalDateTime now = LocalDateTime.now();

        stateMachineMap.entrySet().removeIf(entry -> {
            String key = entry.getKey();
            LocalDateTime lastAccess = lastAccessTimeMap.get(key);

            if (lastAccess == null ||
                Duration.between(lastAccess, now).toMinutes() > idleTimeoutMinutes) {

                StateMachine sm = entry.getValue();
                try {
                    stateMachinePool.returnObject(sm);
                    lastAccessTimeMap.remove(key);
                    log.info("回收空闲状态机: key={}", key);
                    return true;
                } catch (Exception e) {
                    log.error("回收状态机失败", e);
                    return false;
                }
            }
            return false;
        });
    }

    private String generateKey(Long channelId, Long siteId) {
        return siteId + ":" + channelId;
    }
}
```

```java
                event.getChannelId(),
                event.getSiteId(),
                AlarmType.PDI_VIOLATION,
                event.getEndTime(),
                durationResult
            );
            eventPublisher.publishEvent(alarmEvent);
        }

        // 发布作业完成事件
        // 由pdi-behavior-archive模块监听并更新档案
    }
}
```

### 3.4 StateSequenceMatcher (状态序列匹配器)

```java
/**
 * 状态序列匹配器
 * 实现滑动窗口匹配算法，识别进入/离开序列
 */
@Component
@Slf4j
public class StateSequenceMatcher {

    /** 进入序列: S7 → S8 → S3 */
    private static final List<State> ENTRY_SEQUENCE = List.of(State.S7, State.S8, State.S3);

    /** 离开序列: S3 → S8 → S7 → S5/S1 */
    private static final List<State> EXIT_SEQUENCE_PREFIX = List.of(State.S3, State.S8, State.S7);

    /** 离开序列结束状态 */
    private static final Set<State> EXIT_END_STATES = Set.of(State.S5, State.S1);

    /**
     * 匹配进入序列
     * @param stateHistory 历史状态序列
     * @return 匹配结果
     */
    public SequenceMatchResult matchEntrySequence(List<StateSnapshot> stateHistory) {
        if (stateHistory.size() < ENTRY_SEQUENCE.size()) {
            return SequenceMatchResult.notMatched();
        }

        // 取最近3个状态
        List<StateSnapshot> recent = getLastN(stateHistory, 3);

        for (int i = 0; i < ENTRY_SEQUENCE.size(); i++) {
            if (recent.get(i).getState() != ENTRY_SEQUENCE.get(i)) {
                return SequenceMatchResult.notMatched();
            }
        }

        // 匹配成功，返回进入时间 (S3的时间戳)
        LocalDateTime entryTime = recent.get(2).getTimestamp();
        return SequenceMatchResult.matched(entryTime, recent.get(2).getStateStreamId());
    }

    /**
     * 匹配离开序列
     * @param stateHistory 历史状态序列
     * @return 匹配结果
     */
    public SequenceMatchResult matchExitSequence(List<StateSnapshot> stateHistory) {
        if (stateHistory.size() < 4) {
            return SequenceMatchResult.notMatched();
        }

        // 取最近4个状态
        List<StateSnapshot> recent = getLastN(stateHistory, 4);

        // 检查前3个状态是否匹配 S3 → S8 → S7
        for (int i = 0; i < EXIT_SEQUENCE_PREFIX.size(); i++) {
            if (recent.get(i).getState() != EXIT_SEQUENCE_PREFIX.get(i)) {
                return SequenceMatchResult.notMatched();
            }
        }

        // 检查最后一个状态是否为 S5 或 S1
        State lastState = recent.get(3).getState();
        if (!EXIT_END_STATES.contains(lastState)) {
            return SequenceMatchResult.notMatched();
        }

        // 匹配成功，返回离开时间
        LocalDateTime exitTime = recent.get(3).getTimestamp();
        return SequenceMatchResult.matched(exitTime, recent.get(3).getStateStreamId());
    }

    /**
     * 滑动窗口匹配 (用于复杂场景)
     * @param stateHistory 历史状态序列
     * @param pattern 待匹配的模式
     * @return 所有匹配位置
     */
    public List<SequenceMatchResult> slidingWindowMatch(
            List<StateSnapshot> stateHistory,
            List<State> pattern) {

        List<SequenceMatchResult> matches = new ArrayList<>();

        if (stateHistory.size() < pattern.size()) {
            return matches;
        }

        // 滑动窗口
        for (int i = 0; i <= stateHistory.size() - pattern.size(); i++) {
            boolean matched = true;

            for (int j = 0; j < pattern.size(); j++) {
                if (stateHistory.get(i + j).getState() != pattern.get(j)) {
                    matched = false;
                    break;
                }
            }

            if (matched) {
                StateSnapshot lastSnapshot = stateHistory.get(i + pattern.size() - 1);
                matches.add(SequenceMatchResult.matched(
                    lastSnapshot.getTimestamp(),
                    lastSnapshot.getStateStreamId()
                ));
            }
        }

        return matches;
    }

    private List<StateSnapshot> getLastN(List<StateSnapshot> list, int n) {
        int size = list.size();
        if (size <= n) {
            return new ArrayList<>(list);
        }
        return new ArrayList<>(list.subList(size - n, size));
    }
}

/**
 * 序列匹配结果
 */
@Data
@AllArgsConstructor(access = AccessLevel.PRIVATE)
public class SequenceMatchResult {
    private final boolean matched;
    private final LocalDateTime timestamp;
    private final Long stateStreamId;

    public static SequenceMatchResult matched(LocalDateTime timestamp, Long stateStreamId) {
        return new SequenceMatchResult(true, timestamp, stateStreamId);
    }

    public static SequenceMatchResult notMatched() {
        return new SequenceMatchResult(false, null, null);
    }
}
```

```java
/**

 * 支持单通道时长计算和多车门合并计算

@Component
@Slf4j
public class PdiTaskCalculator {

    private final ChannelRepository channelRepository;
    private final SystemConfigRepository configRepository;

    public PdiTaskCalculator(ChannelRepository channelRepository,
                            SystemConfigRepository configRepository) {
        this.channelRepository = channelRepository;
        this.configRepository = configRepository;
    }

    /**
     * 计算单通道作业时长
     */
    public TaskDurationResult calculate(Long channelId,
                                        LocalDateTime entryTime,
                                        LocalDateTime exitTime) {
        Duration duration = Duration.between(entryTime, exitTime);
        long seconds = duration.getSeconds();

        // 获取该通道的标准工时
        Integer standardMinutes = getStandardDuration(channelId);
        long standardSeconds = standardMinutes * 60L;

        // 计算偏差
        long deviationSeconds = seconds - standardSeconds;

        return TaskDurationResult.builder()
            .channelId(channelId)
            .entryTime(entryTime)
            .exitTime(exitTime)
            .actualSeconds(seconds)
            .standardSeconds(standardSeconds)
            .deviationSeconds(deviationSeconds)
            .build();
    }

    /**
     * 多车门合并计算
     * 总时长 = max(T4, T5, T6) - min(T1, T2, T3)
     *
     * @param tasks 各车门作业记录
     * @return 合并后的时长结果
     */
    public TaskDurationResult calculateMerged(List<SingleTaskInfo> tasks) {
        if (tasks == null || tasks.isEmpty()) {
            throw new IllegalArgumentException("任务列表不能为空");
        }

        // 找出最早开始时间和最晚结束时间
        LocalDateTime earliestStart = tasks.stream()
            .map(SingleTaskInfo::getEntryTime)
            .min(Comparator.naturalOrder())
            .orElseThrow();

        LocalDateTime latestEnd = tasks.stream()
            .map(SingleTaskInfo::getExitTime)
            .max(Comparator.naturalOrder())
            .orElseThrow();

        // 计算总时长
        Duration totalDuration = Duration.between(earliestStart, latestEnd);
        long totalSeconds = totalDuration.getSeconds();

        // 获取车辆PDI标准工时 (默认15分钟)
        Integer vehicleStandardMinutes = configRepository
            .getVehiclePdiStandardMinutes()
            .orElse(15);
        long standardSeconds = vehicleStandardMinutes * 60L;

        // 计算偏差
        long deviationSeconds = totalSeconds - standardSeconds;

        return TaskDurationResult.builder()
            .merged(true)
            .subTasks(tasks)
            .entryTime(earliestStart)
            .exitTime(latestEnd)
            .actualSeconds(totalSeconds)
            .standardSeconds(standardSeconds)
            .deviationSeconds(deviationSeconds)
            .build();
    }

    /**
     * 获取通道的标准工时
     */
    private Integer getStandardDuration(Long channelId) {
        Channel channel = channelRepository.findById(channelId)
            .orElseThrow(() -> new EntityNotFoundException("通道不存在"));

        // 根据通道算法类型获取对应标准工时
        String algorithmType = channel.getAlgorithmType();

        return switch (algorithmType) {
            case "pdi_left_front" -> configRepository.getLeftFrontStandardMinutes().orElse(12);
            case "pdi_left_rear" -> configRepository.getLeftRearStandardMinutes().orElse(12);
            case "pdi_slide" -> configRepository.getSlideStandardMinutes().orElse(12);
            default -> 12; // 默认值
        };
    }
}

/**
 * 单任务信息
 */
@Data
@Builder
public class SingleTaskInfo {
    private Long channelId;
    private String channelName;
    private LocalDateTime entryTime;
    private LocalDateTime exitTime;
    private Long durationSeconds;
}

/**
 * 作业时长计算结果
 */
@Data
@Builder
public class TaskDurationResult {
    private Long channelId;
    private boolean merged;                    // 是否为合并计算
    private List<SingleTaskInfo> subTasks;     // 子任务列表 (合并时有效)
    private LocalDateTime entryTime;           // 进入时间
    private LocalDateTime exitTime;            // 离开时间
    private long actualSeconds;                // 实际时长(秒)
    private long standardSeconds;              // 标准时长(秒)
    private long deviationSeconds;             // 偏差时长(秒)

    /**
     * 获取实际时长格式化字符串
     */
    public String getFormattedActualDuration() {
        long minutes = actualSeconds / 60;
        long seconds = actualSeconds % 60;
        return String.format("%d分%d秒", minutes, seconds);
    }

    /**
     * 获取偏差时长格式化字符串
     */
    public String getFormattedDeviation() {
        long minutes = Math.abs(deviationSeconds) / 60;
        long seconds = Math.abs(deviationSeconds) % 60;
        String sign = deviationSeconds >= 0 ? "+" : "-";
        return String.format("%s%d分%d秒", sign, minutes, seconds);
    }
```

### 3.6 AlarmJudge (报警判定器)

```java
/**
 * 报警判定器
 * 根据作业时长判定合规等级
 */
@Component
@Slf4j
public class AlarmJudge {

    /** 临界阈值比例 (0.9 = 90%) */
    @Value("${rule-engine.alarm.critical-ratio:0.9}")
    private double criticalRatio;

    /**
     * 判定合规等级
     *
     * 判定规则:
     * - duration >= standard: 合格 (PASSED)
     * - duration >= 0.9 * standard: 临界 (CRITICAL)
     * - duration < 0.9 * standard: 不合格/报警 (FAILED)
     */
    public ComplianceLevel judge(TaskDurationResult durationResult) {
        long actual = durationResult.getActualSeconds();
        long standard = durationResult.getStandardSeconds();
        long criticalThreshold = (long) (standard * criticalRatio);

        if (actual >= standard) {
            log.debug("作业时长合格: actual={}, standard={}", actual, standard);
            return ComplianceLevel.PASSED;
        } else if (actual >= criticalThreshold) {
            log.debug("作业时长临界: actual={}, criticalThreshold={}", actual, criticalThreshold);
            return ComplianceLevel.CRITICAL;
        } else {
            log.info("作业时长不合格: actual={}, standard={}, deviation={}",
                actual, standard, durationResult.getDeviationSeconds());
            return ComplianceLevel.FAILED;
        }
    }

    /**
     * 判定并生成报警详情
     */
    public AlarmJudgeResult judgeWithDetails(TaskDurationResult durationResult,
                                              Long channelId,
                                              String location) {
        ComplianceLevel level = judge(durationResult);

        String description = generateDescription(durationResult, level, location);

        return AlarmJudgeResult.builder()
            .complianceLevel(level)
            .needAlarm(level == ComplianceLevel.FAILED)
            .alarmType(AlarmType.PDI_VIOLATION)
            .description(description)
            .durationResult(durationResult)
            .build();
    }

    private String generateDescription(TaskDurationResult result,
                                       ComplianceLevel level,
                                       String location) {
        return switch (level) {
            case PASSED -> String.format("%s PDI作业达标，时长: %s (标准: %s)",
                location,
                result.getFormattedActualDuration(),
                formatDuration(result.getStandardSeconds()));
            case CRITICAL -> String.format("%s PDI作业临界，时长: %s (标准: %s, 偏差: %s)",
                location,
                result.getFormattedActualDuration(),
                formatDuration(result.getStandardSeconds()),
                result.getFormattedDeviation());
            case FAILED -> String.format("%s PDI作业未达标，时长: %s (标准: %s, 偏差: %s)",
                location,
                result.getFormattedActualDuration(),
                formatDuration(result.getStandardSeconds()),
                result.getFormattedDeviation());
        };
    }

    private String formatDuration(long seconds) {
        long minutes = seconds / 60;
        long secs = seconds % 60;
        return String.format("%d分%d秒", minutes, secs);
    }
}

/**
 * 合规等级枚举
 */
public enum ComplianceLevel {
    /** 合格 */
    PASSED(1, "合格"),
    /** 临界 */
    CRITICAL(2, "临界"),
    /** 不合格 */
    FAILED(3, "不合格");

    private final int code;
    private final String description;

    ComplianceLevel(int code, String description) {
        this.code = code;
        this.description = description;
    }
}

 * 报警判定结果
 */
@Data
@Builder
public class AlarmJudgeResult {
    private ComplianceLevel complianceLevel;
    private boolean needAlarm;
    private AlarmType alarmType;
    private String description;
    private TaskDurationResult durationResult;
```

## 4. 核心算法设计

### 4.1 状态序列匹配算法 (滑动窗口)

```
算法: SlidingWindowMatch
输入:
    - stateHistory: 历史状态序列 [S1, S2, ..., Sn]
    - pattern: 待匹配模式 [P1, P2, ..., Pm]
输出:
    - matches: 所有匹配位置列表

1. 初始化空列表 matches
2. 如果 n < m, 返回空列表
3. 对于 i 从 0 到 n-m:
   a. 设置 matched = true
   b. 对于 j 从 0 到 m-1:
      - 如果 stateHistory[i+j] != pattern[j]:
          * 设置 matched = false
          * 跳出循环
   c. 如果 matched 为 true:
      - 将匹配结果加入 matches
4. 返回 matches

时间复杂度: O(n * m)
空间复杂度: O(1) (不包括输出)
```

### 4.2 多车门合并算法

```
算法: MergeMultiDoorTasks
输入:
    - tasks: 各车门作业列表 [{channelId, entryTime, exitTime}, ...]
输出:
    - mergedResult: 合并后的作业结果

1. 如果 tasks 为空, 抛出异常
2. 计算 earliestStart = min(task.entryTime for task in tasks)
3. 计算 latestEnd = max(task.exitTime for task in tasks)
4. 计算 totalDuration = latestEnd - earliestStart
5. 获取车辆PDI标准工时 vehicleStandard (默认15分钟)
6. 计算 deviation = totalDuration - vehicleStandard
7. 返回合并结果:
   {
       entryTime: earliestStart,
       exitTime: latestEnd,
       actualDuration: totalDuration,
       standardDuration: vehicleStandard,
       deviation: deviation
   }

示例:
    左前门: T1=10:00, T4=10:08  (8分钟)
    左后门: T2=10:02, T5=10:10  (8分钟)
    滑移门: T3=10:01, T6=10:09  (8分钟)

    合并结果:
    earliestStart = min(10:00, 10:02, 10:01) = 10:00
    latestEnd = max(10:08, 10:10, 10:09) = 10:10
    totalDuration = 10分钟
```

### 4.3 作业时长判定算法

```
算法: JudgeCompliance
输入:
    - actualDuration: 实际作业时长(秒)
    - standardDuration: 标准工时(秒)
    - criticalRatio: 临界比例 (默认0.9)
输出:
    - complianceLevel: 合规等级

1. 计算临界阈值 criticalThreshold = standardDuration * criticalRatio
2. 如果 actualDuration >= standardDuration:
       返回 PASSED (合格)
3. 否则如果 actualDuration >= criticalThreshold:
       返回 CRITICAL (临界)
4. 否则:
       返回 FAILED (不合格, 触发报警)

判定矩阵:
┌─────────────────┬─────────────────┬─────────────────┐
│   actual >= std │ actual >= 0.9*std│   actual < 0.9*std  │
├─────────────────┼─────────────────┼─────────────────┤
│     PASSED      │    CRITICAL     │     FAILED      │
│     (合格)       │    (临界)        │   (报警)        │
└─────────────────┴─────────────────┴─────────────────┘
```
