# 接口文档自动生成器（固定格式）

## 核心功能
你只需要告诉我：
- 接口在文件中的**位置**（第几行、哪一段）
- 或直接粘贴**接口信息**

我会**严格按照你指定的固定格式**，自动生成完整 Markdown 接口文档，格式完全统一、不需要你调整。
