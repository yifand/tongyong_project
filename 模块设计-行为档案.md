## 1. 模块职责简述

行为档案模块（`pdi-behavior-archive`）负责PDI作业行为档案的管理与追溯，是PDI智能监测平台的核心业务模块之一。

### 1.1 核心职责

| 职责 | 说明 |
|------|------|
| **档案列表查询** | 支持按站点、状态、时间范围等多维度筛选查询PDI作业档案 |
| **档案详情查询** | 提供档案完整详情，包含基本信息和时间线数据 |
| **图片包下载** | 支持将档案关联的所有截图打包下载为ZIP文件 |
| **档案状态计算** | 自动计算档案状态：进行中/达标/未达标 |

### 1.2 模块依赖

```
pdi-behavior-archive
    ├── pdi-common (公共基础模块)
    └── pdi-rule-engine (规则引擎模块，获取标准工时配置)
```

### 1.3 数据流转

```
┌─────────────┐     ┌─────────────────┐     ┌─────────────────┐
│  pdi_task   │────→│ behavior_archive│←────│ archive_timeline│
│  (PDI作业)   │     │   (行为档案)     │     │  (档案时间线)    │
└─────────────┘     └────────┬────────┘     └─────────────────┘
                             │
                             ↓
                    ┌─────────────────┐
                    │  REST API 接口   │
                    │  • 列表查询      │
                    │  • 详情查询      │
                    │  • 图片包下载    │
                    └─────────────────┘
```

### 2.4 Service 实现类

#### BehaviorArchiveServiceImpl

```java
@Service
@RequiredArgsConstructor
@Slf4j
public class BehaviorArchiveServiceImpl implements BehaviorArchiveService {

    private final BehaviorArchiveRepository archiveRepository;
    private final ArchiveTimelineService timelineService;
    private final PdiTaskRepository pdiTaskRepository;
    private final ArchiveMapper archiveMapper;

    @Override
    public PageResult<ArchiveResponse> queryArchiveList(ArchiveListRequest request, int page, int size) {
        // 构建查询条件
        QBehaviorArchive qArchive = QBehaviorArchive.behaviorArchive;
        BooleanBuilder predicate = buildQueryPredicate(request, qArchive);

        // 执行分页查询
        Pageable pageable = PageRequest.of(page - 1, size,
                Sort.by(Sort.Direction.DESC, "startTime"));

        Page<BehaviorArchive> archivePage = archiveRepository.findAll(predicate, pageable);

        // 转换并返回
        List<ArchiveResponse> responses = archivePage.getContent().stream()
                .map(archiveMapper::toResponse)
                .collect(Collectors.toList());

        return PageResult.of(responses, archivePage.getTotalElements(), page, size);
    }

    private BooleanBuilder buildQueryPredicate(ArchiveListRequest request, QBehaviorArchive qArchive) {
        BooleanBuilder predicate = new BooleanBuilder();

        // 站点筛选
        if (request.getSiteId() != null) {
            predicate.and(qArchive.siteId.eq(request.getSiteId()));
        }

        // 状态筛选
        if (request.getStatus() != null) {
            predicate.and(qArchive.status.eq(request.getStatus()));
        }

        // 时间范围筛选
        if (request.getStartTimeFrom() != null) {
            predicate.and(qArchive.startTime.goe(request.getStartTimeFrom()));
        }
        if (request.getStartTimeTo() != null) {
            predicate.and(qArchive.startTime.loe(request.getStartTimeTo()));
        }

        // 数据权限：非超级管理员只能查看本站点数据
        Long currentSiteId = SecurityUtils.getCurrentSiteId();
        if (currentSiteId != null) {
            predicate.and(qArchive.siteId.eq(currentSiteId));
        }

        // 排除已删除数据
        predicate.and(qArchive.deletedAt.isNull());

        return predicate;
    }

    @Override
    public ArchiveDetailResponse getArchiveDetail(Long archiveId) {
        // 查询档案
        BehaviorArchive archive = archiveRepository.findById(archiveId)
                .orElseThrow(() -> new ArchiveException("档案不存在: " + archiveId));

        // 数据权限校验
        checkDataPermission(archive.getSiteId());

        // 查询关联的PDI任务
        PdiTask pdiTask = pdiTaskRepository.findById(archive.getPdiTaskId())
                .orElseThrow(() -> new ArchiveException("关联的PDI任务不存在"));

        // 组装详情
        ArchiveDetailResponse detail = archiveMapper.toDetailResponse(archive);

        // 获取时间线
        List<TimelineItemDTO> timeline = timelineService.assembleTimeline(archive, pdiTask);
        detail.setTimeline(timeline);

        return detail;
    }

    @Override
    @Transactional
    public BehaviorArchive createOrUpdateArchive(PdiTask pdiTask) {
        // 查找是否已存在档案
        Optional<BehaviorArchive> existingArchive = archiveRepository
                .findByPdiTaskId(pdiTask.getId());

        BehaviorArchive archive;
        if (existingArchive.isPresent()) {
            archive = existingArchive.get();
            // 更新档案信息
            archive.setEndTime(pdiTask.getEndTime());
            archive.setActualDuration(pdiTask.getActualDuration());
            archive.setStatus(calculateArchiveStatus(pdiTask).getCode());
        } else {
            // 创建新档案
            archive = new BehaviorArchive();
            archive.setPdiTaskId(pdiTask.getId());
            archive.setChannelId(pdiTask.getChannelId());
            archive.setSiteId(pdiTask.getSiteId());
            archive.setStartTime(pdiTask.getStartTime());
            archive.setEndTime(pdiTask.getEndTime());
            archive.setEstimatedDuration(pdiTask.getEstimatedDuration());
            archive.setActualDuration(pdiTask.getActualDuration());
            archive.setStatus(calculateArchiveStatus(pdiTask).getCode());
        }

        return archiveRepository.save(archive);
    }

    @Override
    public ArchiveStatus calculateArchiveStatus(BehaviorArchive archive) {
        // 进行中：end_time为null
        if (archive.getEndTime() == null) {
            return ArchiveStatus.IN_PROGRESS;
        }

        // 计算实际时长（秒）与标准时长（分钟转秒）
        int actualSeconds = archive.getActualDuration() != null ?
                archive.getActualDuration() : 0;
        int standardSeconds = archive.getEstimatedDuration() != null ?
                archive.getEstimatedDuration() * 60 : 0;

        // 达标判定：实际时长 >= 标准时长 * 0.9
        if (standardSeconds > 0 && actualSeconds >= standardSeconds * 0.9) {
            return ArchiveStatus.QUALIFIED;
        } else {
            return ArchiveStatus.UNQUALIFIED;
        }
    }

    private ArchiveStatus calculateArchiveStatus(PdiTask pdiTask) {
        if (pdiTask.getEndTime() == null) {
            return ArchiveStatus.IN_PROGRESS;
        }

        int actualSeconds = pdiTask.getActualDuration() != null ?
                pdiTask.getActualDuration() : 0;
        int standardSeconds = pdiTask.getEstimatedDuration() != null ?
                pdiTask.getEstimatedDuration() * 60 : 0;

        if (standardSeconds > 0 && actualSeconds >= standardSeconds * 0.9) {
            return ArchiveStatus.QUALIFIED;
        } else {
            return ArchiveStatus.UNQUALIFIED;
        }
    }

    private void checkDataPermission(Long siteId) {
        Long currentSiteId = SecurityUtils.getCurrentSiteId();
        if (currentSiteId != null && !currentSiteId.equals(siteId)) {
            throw new ArchiveException("无权访问该站点数据");
        }
    }
}
```

#### ArchiveExportServiceImpl

```java
@Service
@RequiredArgsConstructor
@Slf4j
public class ArchiveExportServiceImpl implements ArchiveExportService {

    private final BehaviorArchiveRepository archiveRepository;
    private final ArchiveTimelineRepository timelineRepository;
    private final StorageService storageService;

    @Override
    public ResponseEntity<StreamingResponseBody> downloadImagePackage(Long archiveId,
                                                                       HttpServletResponse response) {
        // 查询档案
        BehaviorArchive archive = archiveRepository.findById(archiveId)
                .orElseThrow(() -> new ArchiveException("档案不存在: " + archiveId));

        // 查询时间线（获取所有图片URL）
        List<ArchiveTimeline> timelines = timelineRepository
                .findByArchiveIdOrderBySeqAsc(archiveId);

        if (timelines.isEmpty()) {
            throw new ArchiveException("该档案没有可下载的图片");
        }

        // 构建文件名
        String zipFileName = buildZipFileName(archive);

        // 设置响应头
        response.setContentType("application/zip");
        response.setHeader("Content-Disposition",
                "attachment; filename*=UTF-8''" + URLEncoder.encode(zipFileName, StandardCharsets.UTF_8));

        // 使用StreamingResponseBody流式返回
        StreamingResponseBody stream = out -> {
            try (ZipOutputStream zos = new ZipOutputStream(out)) {
                int index = 1;
                for (ArchiveTimeline timeline : timelines) {
                    if (StringUtils.hasText(timeline.getImageUrl())) {
                        addImageToZip(zos, timeline, index++);
                    }
                }
                zos.finish();
            } catch (Exception e) {
                log.error("生成图片包失败, archiveId={}", archiveId, e);
                throw new ArchiveException("图片包生成失败: " + e.getMessage());
            }
        };

        return ResponseEntity.ok(stream);
    }

    private void addImageToZip(ZipOutputStream zos, ArchiveTimeline timeline, int index)
            throws IOException {
        String imageUrl = timeline.getImageUrl();
        String fileName = String.format("%02d_%s_%s.jpg",
                index,
                timeline.getEventTime().format(DateTimeFormatter.ofPattern("HHmmss")),
                sanitizeFileName(timeline.getAction()));

        try (InputStream is = getImageInputStream(imageUrl)) {
            if (is == null) {
                log.warn("无法获取图片: {}", imageUrl);
                return;
            }

            ZipEntry zipEntry = new ZipEntry(fileName);
            zos.putNextEntry(zipEntry);

            // 流式复制，避免内存溢出
            byte[] buffer = new byte[8192];
            int len;
            while ((len = is.read(buffer)) > 0) {
                zos.write(buffer, 0, len);
            }

            zos.closeEntry();
        } catch (Exception e) {
            log.error("添加图片到ZIP失败: {}", imageUrl, e);
        }
    }

    @Override
    public InputStream getImageInputStream(String imageUrl) {
        return storageService.getObject(imageUrl);
    }

    @Override
    public String buildZipFileName(BehaviorArchive archive) {
        String siteCode = getSiteCode(archive.getSiteId());
        String dateStr = archive.getStartTime().format(DateTimeFormatter.ofPattern("yyyyMMdd_HHmmss"));
        return String.format("archive_%s_%s.zip", siteCode, dateStr);
    }

    private String getSiteCode(Long siteId) {
        // 从缓存或数据库获取站点编码
        return "SITE" + siteId;
    }

    private String sanitizeFileName(String input) {
        if (input == null) {
            return "unknown";
        }
        return input.replaceAll("[^a-zA-Z0-9\\u4e00-\\u9fa5]", "_");
    }
}
```

```sql
CREATE TABLE behavior_archive (

    estimated_duration  INTEGER COMMENT '预估检查时间（分钟）',
    actual_duration     INTEGER COMMENT '实际检查时间（秒）',
    status              SMALLINT NOT NULL DEFAULT 0 COMMENT '状态：0-进行中，1-达标，2-未达标',
    created_at          TIMESTAMP NOT NULL DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
    updated_at          TIMESTAMP NOT NULL DEFAULT CURRENT_TIMESTAMP COMMENT '更新时间',
    deleted_at          TIMESTAMP COMMENT '删除时间（逻辑删除）',
```

## 5. 时间线组装逻辑

### 5.1 时间线数据来源

时间线数据从以下两个来源组装：

1. **pdi_task表**: 提供作业开始时间、结束时间
2. **archive_timeline表**: 提供中间过程节点

### 5.2 组装流程

```
┌─────────────────────────────────────────────────────────────┐
│                     时间线组装流程                           │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  输入: archive (behavior_archive) + pdiTask (pdi_task)      │
│                      │                                      │
│                      ↓                                      │
│  ┌─────────────────────────────────────────────┐           │
│  │ 步骤1: 创建起始节点                          │           │
│  │ • 时间: pdiTask.startTime                   │           │
│  │ • 行为: "人员进入车内"                       │           │
│  │ • 类型: START                               │           │
│  │ • 偏移: 0秒                                 │           │
│  └─────────────────────┬───────────────────────┘           │
│                        ↓                                    │
│  ┌─────────────────────────────────────────────┐           │
│  │ 步骤2: 查询中间节点                          │           │
│  │ SELECT * FROM archive_timeline              │           │
│  │ WHERE archive_id = ?                        │           │
│  │ ORDER BY seq ASC                            │           │
│  └─────────────────────┬───────────────────────┘           │
│                        ↓                                    │
│  ┌─────────────────────────────────────────────┐           │
│  │ 步骤3: 创建结束节点（如作业已完成）           │           │
│  │ • 时间: pdiTask.endTime                     │           │
│  │ • 行为: "人员离开，检查结束"                 │           │
│  │ • 类型: END                                 │           │
│  │ • 偏移: (endTime - startTime)秒             │           │
│  └─────────────────────┬───────────────────────┘           │
│                        ↓                                    │
│  ┌─────────────────────────────────────────────┐           │
│  │ 步骤4: 合并排序                              │           │
│  │ • 按 eventTime 升序排列                      │           │
│  │ • 重新计算 seq 序号                          │           │
│  │ • 计算每个节点的 offsetSeconds               │           │
│  └─────────────────────┬───────────────────────┘           │
│                        ↓                                    │
│                 返回 List<TimelineItemDTO>                  │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 5.3 实现代码

```java
@Service
@RequiredArgsConstructor
public class ArchiveTimelineServiceImpl implements ArchiveTimelineService {

    private final ArchiveTimelineRepository timelineRepository;
    private final StateStreamRepository stateStreamRepository;

    @Override
    public List<TimelineItemDTO> assembleTimeline(BehaviorArchive archive, PdiTask pdiTask) {
        List<TimelineItemDTO> timeline = new ArrayList<>();

        LocalDateTime startTime = pdiTask.getStartTime();

        // 1. 添加起始节点
        timeline.add(TimelineItemDTO.builder()
                .seq(1)
                .eventTime(startTime)
                .action("人员进入车内")
                .imageUrl(getEnterImageUrl(pdiTask))
                .nodeType("start")
                .offsetSeconds(0)
                .build());

        // 2. 查询并添加中间节点
        List<ArchiveTimeline> middleNodes = timelineRepository
                .findByArchiveIdAndSeqGreaterThanOrderBySeqAsc(archive.getId(), 1);

        int seq = 2;
        for (ArchiveTimeline node : middleNodes) {
            // 跳过结束节点（seq为最大值）
            if (pdiTask.getEndTime() != null &&
                node.getEventTime().equals(pdiTask.getEndTime())) {
                continue;
            }

            timeline.add(TimelineItemDTO.builder()
                    .seq(seq++)
                    .eventTime(node.getEventTime())
                    .action(node.getAction())
                    .imageUrl(node.getImageUrl())
                    .nodeType("process")
                    .offsetSeconds((int) Duration.between(startTime, node.getEventTime()).getSeconds())
                    .build());
        }

        // 3. 添加结束节点（如果作业已完成）
        if (pdiTask.getEndTime() != null) {
            timeline.add(TimelineItemDTO.builder()
                    .seq(seq)
                    .eventTime(pdiTask.getEndTime())
                    .action("人员离开，检查结束")
                    .imageUrl(getExitImageUrl(pdiTask))
                    .nodeType("end")
                    .offsetSeconds((int) Duration.between(startTime, pdiTask.getEndTime()).getSeconds())
                    .build());
        }

        return timeline;
    }

    private String getEnterImageUrl(PdiTask pdiTask) {
        // 从state_stream获取人员进入时的截图
        StateStream enterState = stateStreamRepository
                .findFirstByChannelIdAndStateCodeAndEventTimeGreaterThanEqualOrderByEventTimeAsc(
                        pdiTask.getChannelId(), 3, pdiTask.getStartTime());
        return enterState != null ? enterState.getImageUrl() : null;
    }

    private String getExitImageUrl(PdiTask pdiTask) {
        // 从state_stream获取人员离开时的截图
        StateStream exitState = stateStreamRepository
                .findFirstByChannelIdAndStateCodeInAndEventTimeLessThanEqualOrderByEventTimeDesc(
                        pdiTask.getChannelId(), List.of(1, 5), pdiTask.getEndTime());
        return exitState != null ? exitState.getImageUrl() : null;
    }
}
```

### 5.4 时间节点类型定义

| 类型 | 说明 | 示例 |
|------|------|------|
| start | 作业开始 | 人员进入车内 |
| process | 作业进行中 | 开始检查作业、检查进行中 |
| end | 作业结束 | 人员离开，检查结束 |

## 6. 图片包下载实现

### 6.1 实现要点

1. **流式处理**: 使用 `StreamingResponseBody` 避免将所有图片加载到内存
2. **边下载边压缩**: 使用 `ZipOutputStream` 流式写入
3. **错误容错**: 单张图片下载失败不影响整体打包

### 6.4 MinIO实现示例

```java
@Service
@RequiredArgsConstructor
@Slf4j
public class MinioStorageService implements StorageService {

    private final MinioClient minioClient;

    @Value("${minio.bucket-name:archives}")
    private String bucketName;

    @Override
    public InputStream getObject(String objectUrl) {
        try {
            String objectName = extractObjectName(objectUrl);
            return minioClient.getObject(
                GetObjectArgs.builder()
                    .bucket(bucketName)
                    .object(objectName)
                    .build()
            );
        } catch (Exception e) {
            log.error("获取对象失败: {}", objectUrl, e);
            return null;
        }
    }

    private String extractObjectName(String objectUrl) {
        // 从完整URL中提取对象名称
        // http://minio:9000/archives/2026-03-25/image.jpg -> 2026-03-25/image.jpg
        URI uri = URI.create(objectUrl);
        String path = uri.getPath();
        // 去掉bucket名称前缀
        if (path.startsWith("/" + bucketName + "/")) {
            return path.substring(bucketName.length() + 2);
        }
        return path.substring(1);
    }
}
```
