## 1. 模块职责简述

设备管理模块（pdi-device-management）负责平台边缘计算设备的统一管理，包括边缘盒子管理、通道管理、设备状态监控及心跳处理。

### 1.1 核心职责

| 职责项 | 说明 |
|--------|------|
| 边缘盒子管理 | 边缘盒子的CRUD、状态监控、远程重启操作 |
| 通道管理 | 视频通道的CRUD、算法配置、RTSP地址管理 |
| 设备状态监控 | 提供设备状态大屏数据接口，包括在线/离线统计、资源使用率 |
| 心跳处理 | 接收边缘盒子上报的心跳，更新设备状态，检测心跳超时 |

### 1.2 模块依赖

```
pdi-device-management
    ├── pdi-common (公共基础模块)
    └── pdi-algorithm-inlet (接收心跳数据)
```

### 3.1 edge_box 表（边缘盒子表）

```sql
-- 边缘盒子表
CREATE TABLE edge_box (
    id                      BIGSERIAL PRIMARY KEY COMMENT '主键ID',
    site_id                 BIGINT NOT NULL COMMENT '所属站点ID',
    name                    VARCHAR(64) NOT NULL COMMENT '盒子名称',
    ip_address              VARCHAR(15) NOT NULL COMMENT 'IP地址',
    status                  SMALLINT NOT NULL DEFAULT 0 COMMENT '状态: 0-离线, 1-在线',
    last_heartbeat_at       TIMESTAMP COMMENT '最后心跳时间',
    version                 VARCHAR(32) COMMENT '软件版本号',
    cpu_usage               DECIMAL(5,2) COMMENT 'CPU使用率(%)',
    memory_usage            DECIMAL(5,2) COMMENT '内存使用率(%)',
    disk_usage              DECIMAL(5,2) COMMENT '磁盘使用率(%)',
    created_at              TIMESTAMP NOT NULL DEFAULT CURRENT_TIMESTAMP COMMENT '创建时间',
    updated_at              TIMESTAMP NOT NULL DEFAULT CURRENT_TIMESTAMP COMMENT '更新时间',
    deleted_at              TIMESTAMP COMMENT '删除时间(逻辑删除)',
    created_by              BIGINT COMMENT '创建人ID'
);

-- 注释
COMMENT ON TABLE edge_box IS '边缘计算盒子表';

-- 索引
CREATE INDEX idx_edge_box_site_id ON edge_box(site_id) WHERE deleted_at IS NULL;
CREATE INDEX idx_edge_box_status ON edge_box(status) WHERE deleted_at IS NULL;
CREATE INDEX idx_edge_box_heartbeat ON edge_box(last_heartbeat_at) WHERE deleted_at IS NULL;
CREATE INDEX idx_edge_box_site_status ON edge_box(site_id, status) WHERE deleted_at IS NULL;

-- 唯一约束：站点内盒子名称唯一
CREATE UNIQUE INDEX uk_edge_box_site_name ON edge_box(site_id, name) WHERE deleted_at IS NULL;
```

## 5. 心跳处理逻辑

### 5.1 心跳接收流程

```
┌─────────────┐     ┌──────────────────┐     ┌──────────────────┐
│  边缘盒子    │────→│ Algorithm Inlet  │────→│ HeartbeatEvent   │
│  (定时上报)  │     │  (HTTP Receiver) │     │  (Spring Event)  │
└─────────────┘     └──────────────────┘     └────────┬─────────┘
                                                      │
                                                      ↓
                                             ┌──────────────────┐
                                             │ HeartbeatProcessor│
                                             │  (心跳处理器)      │
                                             └────────┬─────────┘
                                                      │
                          ┌───────────────────────────┼───────────────────────────┐
                          ↓                           ↓                           ↓
                   ┌─────────────┐            ┌─────────────┐            ┌─────────────┐
                   │ 验证盒子存在 │            │ 更新在线状态 │            │ 更新资源数据 │
                   └─────────────┘            └─────────────┘            └─────────────┘
```

### 5.2 心跳处理器实现

```java
@Component
@Slf4j
public class HeartbeatProcessor {

    @Autowired
    private EdgeBoxRepository edgeBoxRepository;

    @Autowired
    private DeviceConfig deviceConfig;

    /**
     * 处理心跳事件
     */
    @EventListener
    @Transactional
    public void handleHeartbeat(HeartbeatEvent event) {
        log.debug("收到心跳: boxId={}, timestamp={}",
                  event.getBoxId(), event.getTimestamp());

        // 1. 查询盒子
        EdgeBox box = edgeBoxRepository.findById(event.getBoxId())
            .orElse(null);

        if (box == null || box.getDeletedAt() != null) {
            log.warn("心跳对应的盒子不存在或已删除: boxId={}", event.getBoxId());
            return;
        }

        // 2. 更新盒子状态
        box.setStatus(1);  // 在线
        box.setLastHeartbeatAt(event.getTimestamp());

        // 3. 更新资源使用率
        if (event.getCpuUsage() != null) {
            box.setCpuUsage(event.getCpuUsage());
        }
        if (event.getMemoryUsage() != null) {
            box.setMemoryUsage(event.getMemoryUsage());
        }
        if (event.getDiskUsage() != null) {
            box.setDiskUsage(event.getDiskUsage());
        }
        if (event.getVersion() != null) {
            box.setVersion(event.getVersion());
        }

        edgeBoxRepository.save(box);
        log.debug("心跳处理完成: boxId={}", event.getBoxId());
    }

    /**
     * 定时检测心跳超时
     * 每分钟执行一次
     */
    @Scheduled(fixedRate = 60000)
    @Transactional
    public void checkHeartbeatTimeout() {
        // 计算超时时间
        LocalDateTime timeoutTime = LocalDateTime.now()
            .minusSeconds(deviceConfig.getHeartbeatTimeout());

        // 查询超时盒子
        List<EdgeBox> timeoutBoxes = edgeBoxRepository
            .findTimeoutBoxes(timeoutTime);

        if (CollectionUtils.isEmpty(timeoutBoxes)) {
            return;
        }

        log.info("检测到{}个盒子心跳超时", timeoutBoxes.size());

        // 更新为离线状态
        for (EdgeBox box : timeoutBoxes) {
            box.setStatus(0);  // 离线
            edgeBoxRepository.save(box);

            // 可选：发布盒子离线事件，用于报警通知
            // eventPublisher.publishEvent(new BoxOfflineEvent(box));
        }
    }
}
```

```java
@Service
public class ChannelServiceImpl implements ChannelService {

    /**
     * 验证算法类型有效性
     */
    private void validateAlgorithmType(String algorithmType) {
        if (StringUtils.isBlank(algorithmType)) {
            return;
        }

        AlgorithmType type = AlgorithmType.fromCode(algorithmType);
        if (type == null) {
            throw new DeviceException("无效的算法类型: " + algorithmType);
        }
    }

    @Override
    @Transactional
    public Long createChannel(ChannelRequest request, Long currentSiteId, Long currentUserId) {
        // 验证算法类型
        validateAlgorithmType(request.getAlgorithmType());

        // 验证盒子存在且属于当前站点
        EdgeBox box = edgeBoxRepository.findById(request.getBoxId())
            .orElseThrow(() -> new DeviceException("盒子不存在"));

        if (!box.getSiteId().equals(currentSiteId)) {
            throw new DeviceException("无权访问该盒子");
        }

        // 创建通道
        Channel channel = new Channel();
        channel.setBoxId(request.getBoxId());
        channel.setSiteId(currentSiteId);
        channel.setName(request.getName());
        channel.setType(request.getType());
        channel.setAlgorithmType(request.getAlgorithmType());
        channel.setRtspUrl(request.getRtspUrl());  // 会自动加密
        channel.setStatus(0);
        channel.setCreatedBy(currentUserId);

        channelRepository.save(channel);

        log.info("创建通道成功: channelId={}, name={}, algorithm={}",
                 channel.getId(), channel.getName(), channel.getAlgorithmType());

        return channel.getId();
    }
}
```

### 6.4 RTSP地址脱敏显示

```java
@Component
public class ChannelMapper {

    /**
     * 将RTSP地址脱敏显示
     * rtsp://admin:password@192.168.1.101:554/stream1
     * → rtsp://***:***@***.***.***.***:***/****
     */
    public String maskRtspUrl(String rtspUrl) {
        if (StringUtils.isBlank(rtspUrl)) {
            return null;
        }

        try {
            // 使用正则表达式脱敏
            return rtspUrl.replaceAll("//[^:]+:[^@]+@", "//***:***@")
                         .replaceAll("\\d+\\.\\d+\\.\\d+\\.\\d+", "***.***.***.***")
                         .replaceAll(":\\d+", ":***")
                         .replaceAll("/@.*", "/****");
        } catch (Exception e) {
            return "rtsp://***";
        }
    }
}
```
